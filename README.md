Project Title: 8 Queens Puzzle Game

Description: This project is an interactive web application built using Python Flask that allows users to solve the classic Eight Queens Puzzle. Users can place queens on a chessboard grid, ensuring that each placement adheres to the rules of chess regarding queen movements. The application provides real-time validation and feedback.

Features:

Interactive 8x8 chessboard grid.
Users can click on a cell to place a queen.
Real-time validation to ensure no two queens can attack each other.
Ability to move previously placed queens.
Option to remove a queen with a double-click.
Success message once all 8 queens are placed appropriately.
User-friendly interface designed for first-time users.
Robust error handling to prevent crashes and provide user guidance.

Technologies Used:

Python 3
Flask
HTML/CSS
JavaScript 

Installation Instructions:

Clone the repository: git clone git@github.com:Nidhishree-S-P/8-Queens.git

Navigate to the project directory: cd 8-Queens

Install the required packages: pip install flask

Run the Flask application: python app.py

Open your web browser and go to http://127.0.0.1:5000 to view the application.
